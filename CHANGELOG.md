## 0.1.0

Initial release.

### Added
- VS Code extension with local HTTP bridge for notebook sessions
- MCP server with stdio transport
- Shared types, schemas, and validation library
- 17 MCP tools for notebook discovery, editing, and execution
- 2 MCP prompts: notebook-cite and notebook-review
- Loopback-only bridge with ephemeral bearer token authentication
- Structured logging with pino
- Output handling for text, markdown, JSON, HTML, and image types
- CI workflow with GitHub Actions
- Documentation: architecture, security, development, release, manual verification
